017    /**
018     * This class utilizes functionality of the Java class <code>java.lang.reflect</code>. Specifically, this reflection tool is designed to
019     * perform advanced method or constructor lookups, using a combination of {@link LookupMode} strategies.
020     * <p>
021     * Aside from that there are some nifty methods for things such as:
022     * <ul>
023     * <li>finding out if a given string matches a package name ({@link #isPackage(String)}),</li>
024     * <li>returning the widest number type that will fit a any number on a given list of numbers ({@link #widestNumberClass(Number...)}),</li>
025     * <li>a method to autobox a given value to its counterpart ({@link #autobox(Class)}),</li>
026     * <li>A couple of methods to resolve a direct property of a given object (though they may move to {@link FieldUtils}),</li>
027     * <li>Various handy methods for collecting information on a given <code>Object</code>, such as list of methods, properties or types</li>
028     * <li>An advanced <code>Class</code> lookup ({@link #locateClass(String, boolean, ExternalClassLoader)}), that allows a
029     * full scan (to try all packages known) and an optional {@link ExternalClassLoader} instance that is able to actually compile a .java file
030     * on the fly and load its compile .class file</li>
031     * </ul>
032     * <strong>About the method / constructor lookup facility:</strong>
033     * <p>
034     * An expanded version of <code>getConstructor</code> is implemented that tries to find a constructor of a given datatype, with a given
035     * argument datatypelist, where types do not have to match formal types (auto-boxing, supertypes, implemented interfaces and type
036     * conversions are allowed as they are included in the lookup cycles). This expanded version tries a simple call first (exact match, which
037     * is provided natively by the Java) and when this fails, it generates a list of datatype arrays (signatures) with all possible versions of
038     * any type in the original list possible, and combinations thereof.
039     * <p>
040     * <strong>Observe the following (trivial) example:</strong>
041     * 
042     * <pre>
043     *      interface Foo {
044     *              void foo(Double value, Fruit fruit, char c);
045     *      }
046     *      abstract class A implements Foo {
047     *      }
048     *      abstract class B extends A {
049     *      }
050     * 
051     *      JReflect.findCompatibleJavaMethod(B.class, "foo", EnumSet.allOf(LookupMode.class), double.class, Pear.class, String.class)}
052     * </pre>
053     * 
054     * In the above example, the method foo will be found by finding all methods named "Foo" on the interfaces implemented by supertype
055     * <code>A</code>, and then foo's method signature will be matched using autoboxing on the <code>double</code> type, a cast to the
056     * <code>Fruit</code> supertype for the <code>Pear</code> type and finally by attempting a common conversion from <code>String</code> to
057     * <code>char</code>. This will give you a Java {@link Method}, but you won't be able to invoke it if it was found using a less strict
058     * lookup than one with a simple exact match. There are two ways to do this: use
059     * {@link #invokeCompatibleMethod(Object, Class, String, Object...)} instead or perform the conversion yourself using
060     * {@link ValueConverter#convert(Object[], Class[], boolean)} prior to invoking the method. <code>
061                            ValueConverter.convert(args, method.getParameterTypes())</code>.
062     * <p>
063     * // TODO fix comment about switching the cache<br/>
064     * Because this lookup is potentially very expensive, a cache is present to store lookup results. This cache is turned on/off using the
065     * constructor.
066     * <p>
067     * <strong>Types that are candidates for Autoboxing:</strong>
068     * <ul>
069     * <li><strong>boolean</strong> <code>java.lang.Boolean</code></li>
070     * <li><strong>char</strong> <code>java.lang.Character</code></li>
071     * <li><strong>byte</strong> <code>java.lang.Byte</code></li>
072     * <li><strong>short</strong> <code>java.lang.Short</code></li>
073     * <li><strong>int</strong> <code>java.lang.Integer</code></li>
074     * <li><strong>long</strong> <code>java.lang.Long</code></li>
075     * <li><strong>float</strong> <code>java.lang.Float</code></li>
076     * <li><strong>double</strong> <code>java.lang.Double</code></li>
077     * </ul>
078     * <p>
079     * For types that are candidates for common conversion, please see {@link ValueConverter}.
080     * 
081     * @author Benny Bottema
082     * @see ValueConverter
083     * @see FieldUtils
084     * @see ExternalClassLoader FIXME make JReflect cache optional
085     */
086    public final class JReflect {

088            /**
089             * Defines to allowed lookup modes for matching Java methods and constructors. Each time a lookup failed on signature type, a less
090             * strict lookup is performed, in the following order (signature means: the list of parameters defined for a method or constructor):
091             * <ol>
092             * <li><strong>exact matching</strong>: the given type should exactly match the found types during lookup. This lookup cycle is always
093             * performed first.</li>
094             * <li><strong>autobox matching</strong>: the given type should match its boxed/unboxed version.</li>
095             * <li><strong>polymorphic interface matching</strong>: the given type should match one of the implemented interfaces</li>
096             * <li><strong>polymorphic superclass matching</strong>: the given type should match one of the super classes (for each superclass, the
097             * cycle is repeated, so exact and interface matching come first again before the next superclass up in the chain)</li>
098             * <li><strong>common conversion matching</strong>: if all other lookups fail, one last resort is to try to convert the given type, if a
099             * common type, to any other common type and then try to find a matching method or constructor. See {@link ValueConverter} for more on
100             * the possibilities.</li>
101             * </ol>
102             * 
103             * @author Benny Bottema
104             */
105            public enum LookupMode {
106                    /**
107                     * Indicates that looking for methods includes trying to find compatible signatures by autoboxing the specified arguments.
108                     */
109                    AUTOBOX,
110                    /**
111                     * Indicates that looking for methods includes trying to find compatible signatures by casting the specified arguments to a super
112                     * type.
113                     */
114                    CAST_TO_SUPER,
115                    /**
116                     * Indicates that looking for methods includes trying to find compatible signatures by casting the specified arguments to an
117                     * implemented interface.
118                     */
119                    CAST_TO_INTERFACE,
120                    /**
121                     * Indicates that looking for methods includes trying to find compatible signatures by automatically convert the specified
122                     * arguments.
123                     */
124                    COMMON_CONVERT;
125            }

164            /**
165             * Searches the JVM and optionally all of its packages
166             * 
167             * @param className The name of the class to locate.
168             * @param fullscan Whether a full scan through all available java packages is required.
169             * @param classLoader Optional user-provided classloader.
170             * @return The <code>Class</code> reference if found or <code>null</code> otherwise.
171             */
172            public static Class<?> locateClass(final String className, final boolean fullscan, final ExternalClassLoader classLoader) {
173                    if (fullscan) {
174                            Class<?> _class = null;
175                            // cycle through all packages and try allocating dynamically
176                            final Package[] ps = Package.getPackages();
177                            for (int i = 0; i < ps.length && _class == null; i++) {
178                                    _class = locateClass(ps[i].getName() + "." + className, classLoader);
179                            }
180                            return _class;
181                    } else {
182                            // try standard package used for most common classes
183                            return locateClass("java.lang." + className, classLoader);
184                    }
185            }

187            /**
188             * This function dynamically tries to locate a class. First it searches the class-cache list, then it tries to get it from the Virtual
189             * Machine using {@code Class.forName(String)}.
190             * 
191             * @param fullClassName The <code>Class</code> that needs to be found.
192             * @param classLoader Optional user-provided classloader.
193             * @return The {@code Class} object found from cache or VM.
194             */
195            public static Class<?> locateClass(final String fullClassName, final ExternalClassLoader classLoader) {
196                    try {
197                            Class<?> _class = null;
198                            // /CLOVER:OFF
199                            if (classLoader != null) {
200                                    _class = classLoader.loadClass(fullClassName);
201                            }
202                            // /CLOVER:ON
203                            if (_class == null) {
204                                    _class = Class.forName(fullClassName);
205                            }
206                            return _class;
207                    } catch (final ClassNotFoundException e) {
208                            return null;
209                    }
210            }

212            /**
213             * Simply calls {@link Class#newInstance()} and hides the exception handling boilerplate code.
214             * 
215             * @param _class The datatype for which we need to create a new instance of.
216             * @param <T> Type used to parameterize the return instance.
217             * @return A new parameterized instance of the given type.
218             */
219            public static <T> T newInstanceSimple(final Class<T> _class) {
220                    // /CLOVER:OFF
221                    try {
222                            // /CLOVER:ON
223                            return _class.getConstructor().newInstance();
224                            // /CLOVER:OFF
225                    } catch (IllegalArgumentException e) {
226                            assert false : "we don't pass in arguments";
227                            throw new RuntimeException("unable to invoke parameterless constructor", e);
228                    } catch (SecurityException e) {
229                            throw new RuntimeException("unable to invoke parameterless constructor; security problem", e);
230                    } catch (InstantiationException e) {
231                            throw new RuntimeException("unable to complete instantiation of object", e);
232                    } catch (IllegalAccessException e) {
233                            throw new RuntimeException("unable to access parameterless constructor", e);
234                    } catch (InvocationTargetException e) {
235                            throw new RuntimeException("unable to invoke parameterless constructor", e);
236                    } catch (NoSuchMethodException e) {
237                            throw new RuntimeException("unable to find parameterless constructor (not public?)", e);
238                    }
239                    // /CLOVER:ON
240            }

242            /**
243             * Locates a method on an Object using serveral searchmodes for optimization. First of all a {@link Method} cache is being maintained to
244             * quickly fetch heavily used methods. If not cached before and if a simple search (autoboxing and supertype casts) fails a more complex
245             * search is done where all interfaces are searched for the method as well. If this fails as well, this method will try to autoconvert
246             * the types of the arguments and find a matching signature that way.
247             * 
248             * @param context The object to call the method from (can be null).
249             * @param datatype The class to find the method on.
250             * @param identifier The name of the method to locate.
251             * @param args A list of [non-formal] arguments.
252             * @return The return value of the invoke method, if succesful.
253             * @throws NoSuchMethodException
254             * @throws IllegalArgumentException
255             * @throws IllegalAccessException
256             * @throws InvocationTargetException
257             */
258            public static Object invokeCompatibleMethod(final Object context, final Class<?> datatype, final String identifier,
259                            final Object... args)
260                            throws NoSuchMethodException, IllegalArgumentException, IllegalAccessException, InvocationTargetException {
261                    // determine the signature we want to find a compatible java method for
262                    final Class<?>[] signature = JReflect.collectTypes(args);
263    
264                    // setup lookup procedure starting with simple search mode
265                    EnumSet<LookupMode> lookupMode = EnumSet.of(LookupMode.AUTOBOX, LookupMode.CAST_TO_SUPER);
266                    Method method;
267    
268                    // try to fina a compatible Java method using various lookup modes
269                    try {
270                            method = findCompatibleMethod(datatype, identifier, lookupMode, signature);
271                    } catch (final NoSuchMethodException e1) {
272                            try {
273                                    // moderate search mode
274                                    lookupMode.add(LookupMode.CAST_TO_INTERFACE);
275                                    method = findCompatibleMethod(datatype, identifier, lookupMode, signature);
276                            } catch (final NoSuchMethodException e2) {
277                                    // full searchmode
278                                    lookupMode.add(LookupMode.COMMON_CONVERT);
279                                    method = findCompatibleMethod(datatype, identifier, lookupMode, signature);
280                            }
281                    }
282    
283                    // try to invoke the method with unconverted arguments or convert them if needed
284                    try {
285                            return method.invoke(context, args);
286                    } catch (final IllegalArgumentException e) {
287                            final Object[] convertedArgs = ValueConverter.convert(args, method.getParameterTypes(), true);
288                            return method.invoke(context, convertedArgs);
289                    }
290            }

311            /**
312             * Locates and invokes a {@link Constructor}, using a customized typelist. Avoids dynamically trying to find correct parameter type
313             * list. Can also be used to force up/down casting (ie. passing a specific type of <code>List</code> into a generic type)
314             * 
315             * @param <T> Used to parameterize the returned object so that the caller doesn't need to cast.
316             * @param datatype The class to find the constructor for.
317             * @param signature The typelist used to find correct constructor.
318             * @param args A list of [non-formal] arguments.
319             * @return The instantiated object of class datatype.
320             * @throws IllegalAccessException
321             * @throws InvocationTargetException
322             * @throws InstantiationException
323             * @throws NoSuchMethodException
324             * @see java.lang.reflect.Constructor#newInstance(Object[])
325             */
326            public static <T> T invokeConstructor(final Class<T> datatype, final Class<?>[] signature, final Object[] args)
327                            throws NoSuchMethodException, IllegalAccessException, InvocationTargetException, InstantiationException {
328                    // setup lookup procedure
329                    EnumSet<LookupMode> lookupMode = EnumSet.of(LookupMode.AUTOBOX, LookupMode.CAST_TO_SUPER);
330                    Constructor<T> constructor;
331    
332                    // try to find a compatible Java constructor
333                    try {
334                            // simple search mode
335                            constructor = findCompatibleConstructor(datatype, lookupMode, signature);
336                    } catch (final NoSuchMethodException e1) {
337                            try {
338                                    // moderate search mode
339                                    lookupMode.add(LookupMode.CAST_TO_INTERFACE);
340                                    constructor = findCompatibleConstructor(datatype, lookupMode, signature);
341                            } catch (final NoSuchMethodException e2) {
342                                    // full searchmode
343                                    lookupMode.add(LookupMode.COMMON_CONVERT);
344                                    constructor = findCompatibleConstructor(datatype, lookupMode, signature);
345                            }
346                    }
347    
348                    // try to invoke the constructor with unconverted arguments or convert them if needed
349                    try {
350                            return constructor.newInstance(args);
351                    } catch (final IllegalArgumentException e) {
352                            final Object[] convertedArgs = ValueConverter.convert(args, constructor.getParameterTypes(), true);
353                            return constructor.newInstance(convertedArgs);
354                    }
355            }

357            /**
358             * Creates a new array of class objects harvested from an array of objects.
359             * <p>
360             * NOTE: this method will never return primitive classes (such as double.class, as you can't put primitive values into an array of
361             * Objects (they will be autoboxes by the JVM).
362             * 
363             * @param objects The array of objects to harvest classtypes from.
364             * @return The array with the harvested classtypes.
365             */
366            public static Class<?>[] collectTypes(final Object[] objects) {
367                    // collect classtypes of the arguments
368                    final Class<?>[] types = new Class[objects.length];
369                    for (int i = 0; i < objects.length; i++) {
370                            final Object o = objects[i];
371                            types[i] = o != null ? o.getClass() : Object.class;
372                    }
373                    return types;
374            }

376            /**
377             * Tries to find a {@link Constructor} of a given type, with a given typelist, where types do not match due to formal types <!=>
378             * simpletypes. This expanded version tries a simple call first and when it fails, it generates a list of type arrays with all possible
379             * (un)wraps of any type in the original list possible, and combinations thereof.
380             * 
381             * @param <T> Used to parameterize the returned constructor.
382             * @param datatype The class to get the constructor from.
383             * @param lookupMode Flag indicating the search steps that need to be done.
384             * @param types The list of types as specified by the user.
385             * @return The constructor if found, otherwise exception is thrown.
386             * @exception NoSuchMethodException
387             */
388            @SuppressWarnings("unchecked")
389            public static <T> Constructor<T> findCompatibleConstructor(final Class<T> datatype, final EnumSet<LookupMode> lookupMode,
390                            final Class<?>... types)
391                            throws NoSuchMethodException {
392                    // first try to find the constructor in the method cache
393                    Constructor<T> constructor = (Constructor<T>) getMethodFromCache(datatype, datatype.getName(), types);
394                    if (constructor != null) {
395                            return constructor;
396                    } else {
397                            try {
398                                    // try standard call
399                                    constructor = datatype.getConstructor(types);
400                            } catch (final NoSuchMethodException e) {
401                                    // failed, try all possible wraps/unwraps
402                                    final List<Class<?>[]> typeslist = generateCompatibleSignatures(lookupMode, types);
403                                    for (final Class<?>[] aTypeslist : typeslist) {
404                                            try {
405                                                    constructor = datatype.getConstructor(aTypeslist);
406                                                    break;
407                                            } catch (final NoSuchMethodException x) {
408                                                    // do nothing
409                                            }
410                                    }
411                            }
412                    }
413    
414                    // if a constructor was found (and it wasn't in the cache, because method would've returned already)
415                    if (constructor != null) {
416                            addMethodToCache(datatype, datatype.getName(), constructor, types);
417                            return constructor;
418                    } else {
419                            throw new NoSuchMethodException();
420                    }
421            }

441            /**
442             * Same as <code>getConstructor()</code>, except for getting a {@link Method} of a classtype, using the name to indicate which method
443             * should be located.
444             * 
445             * @param datatype The class to get the constructor from.
446             * @param methodName The name of the method to retrieve from the class.
447             * @param lookupMode Flag indicating the search steps that need to be done.
448             * @param signature The list of types as specified by the user.
449             * @return The method if found, otherwise exception is thrown.
450             * @exception NoSuchMethodException
451             */
452            public static Method findCompatibleMethod(final Class<?> datatype, final String methodName, final EnumSet<LookupMode> lookupMode,
453                            final Class<?>... signature)
454                            throws NoSuchMethodException {
455                    // first try to find the method in the method cache
456                    Method method = (Method) getMethodFromCache(datatype, methodName, signature);
457                    if (method != null) {
458                            return method;
459                    } else {
460                            try {
461                                    // try standard call
462                                    method = getMethod(datatype, methodName, signature);
463                            } catch (final NoSuchMethodException e) {
464                                    // failed, try all possible wraps/unwraps
465                                    final List<Class<?>[]> signatures = generateCompatibleSignatures(lookupMode, signature);
466                                    for (final Class<?>[] compatibleSignature : signatures) {
467                                            try {
468                                                    method = getMethod(datatype, methodName, compatibleSignature);
469                                                    break;
470                                            } catch (final NoSuchMethodException x) {
471                                                    // do nothing
472                                            }
473                                    }
474                            }
475                    }
476    
477                    // if a method was found (and it wasn't in the cache, because method would've returned already)
478                    if (method != null) {
479                            addMethodToCache(datatype, methodName, method, signature);
480                            return method;
481                    } else {
482                            throw new NoSuchMethodException();
483                    }
484            }

486            /**
487             * Searches a specific class object for a {@link Method} using java reflect using a specific signature. This method will first search
488             * all implemented interfaces for the method to avoid visiblity problems.<br />
489             * <br />
490             * An example of such a problem is the <code>Iterator</code> as implemented by the <code>ArrayList</code>. The Iterator is implemented
491             * as a private innerclass and as such not accessible by java reflect (even though the implemented methods are declared <i>public</i>),
492             * unlike the interface's definition.
493             * 
494             * @param datatype The class reference to locate the method on.
495             * @param name The name of the method to find.
496             * @param signature The signature the method should match.
497             * @return The Method found on the datatype that matched the specified signature.
498             * @throws NoSuchMethodException
499             * @see java.lang.Class#getMethod(String, Class[])
500             */
501            public static Method getMethod(final Class<?> datatype, final String name, final Class<?>... signature)
502                            throws NoSuchMethodException {
503                    for (final Class<?> iface : datatype.getInterfaces()) {
504                            try {
505                                    return iface.getMethod(name, signature);
506                            } catch (final NoSuchMethodException e) {
507                                    // do nothing
508                            }
509                    }
510                    return datatype.getMethod(name, signature);
511            }

517             * @param signature The list with original user specified types.
518             * @return The list with converted type-arrays.
519             */

521                    final List<Class<?>[]> signatures = new ArrayList<Class<?>[]>();
522                    generateCompatibleSignatures(0, lookupMode, signatures, signature);
523                    return signatures;

526            /**
527             * Recursively generates a complete list of all possible (un)wraps (autoboxing), supertypes, implemented interfaces, type conversions
528             * and any combination thereof with the specified signature's elements (the individual parameter types).<br />
529             * <br />
530             * The combination signatures are generated in the following order:
531             * <ol>
532             * <li>no conversion; highest priority as it comes closest to user's requirement/specification</li>
533             * <li>autoboxing; the autoboxed counterversion comes closest to the original datatype</li>
534             * <li>interface; where methods can't be found using original type, interface placeholders are attempted</li>
535             * <li>supertype; where methods can't be found using implemented interfaces, supertype placeholders are attempted</li>
536             * <li>conversions; if all else fails, try to convert the datatype for common types (ie. int to String)</li>
537             * </ol>
538             * 
539             * @param index The current index to start mutating from.
540             * @param lookupMode Flag indicating the search steps that need to be done.
541             * @param signatures The central storage list for new type-arrays.
542             * @param signature The list with current types, to mutate further upon.
543             */
544            private static void generateCompatibleSignatures(final int index, final EnumSet<LookupMode> lookupMode,
545                            final List<Class<?>[]> signatures, final Class<?>... signature) {
546                    // if new type array is completed
547                    if (index == signature.length) {
548                            signatures.add(signature);
549                    } else {
550                            // generate new array of types
551                            final Class<?> original = signature[index];
552    
553                            // 1. don't generate compatible list; just try the normal type first
554                            // remember, in combinations types should be allowed to be converted)
555                            generateCompatibleSignatures(index + 1, lookupMode, signatures, signature.clone());
556    
557                            // 2. generate type in which the original can be (un)wrapped
558                            if (lookupMode.contains(LookupMode.AUTOBOX)) {
559                                    final Class<?> autoboxed = autobox(original);
560                                    if (autoboxed != null) {
561                                            final Class<?>[] newSignature = replaceInArray(signature.clone(), index, autoboxed);
562                                            generateCompatibleSignatures(index + 1, lookupMode, signatures, newSignature);
563                                    }
564                            }
565    
566                            // autocast to supertype or interface?
567                            if (lookupMode.contains(LookupMode.CAST_TO_INTERFACE)) {
568                                    // 3. generate implemented interfaces the original value could be converted (cast) into
569                                    for (final Class<?> iface : original.getInterfaces()) {
570                                            final Class<?>[] newSignature = replaceInArray(signature.clone(), index, iface);
571                                            generateCompatibleSignatures(index + 1, lookupMode, signatures, newSignature);
572                                    }
573                            }
574    
575                            if (lookupMode.contains(LookupMode.CAST_TO_SUPER)) {
576                                    // 4. generate supertypes the original value could be converted (cast) into
577                                    Class<?> supertype = original;
578                                    while ((supertype = supertype.getSuperclass()) != null) {
579                                            final Class<?>[] newSignature = replaceInArray(signature.clone(), index, supertype);
580                                            generateCompatibleSignatures(index + 1, lookupMode, signatures, newSignature);
581                                    }
582                            }
583    
584                            // 5. generate types the original value could be converted into
585                            if (lookupMode.contains(LookupMode.COMMON_CONVERT)) {
586                                    for (final Class<?> convert : ValueConverter.collectCompatibleTypes(original)) {
587                                            final Class<?>[] newSignature = replaceInArray(signature.clone(), index, convert);
588                                            generateCompatibleSignatures(index + 1, lookupMode, signatures, newSignature);
589                                    }
590                            }
591                    }
592            }

594            /**
595             * Emulates Java's Autoboxing feature; tries to convert a type to its (un)wrapped counter version.
596             * 
597             * @param c The datatype to convert (autobox).
598             * @return The converted version of the specified type, or null.
599             */
600            public static Class<?> autobox(final Class<?> c) {
601                    // integer
602                    if (c == Integer.class) {
603                            return int.class;
604                    } else if (c == int.class) {
605                            return Integer.class;
606                    } else if (c == Boolean.class) {
607                            return boolean.class;
608                    } else if (c == boolean.class) {
609                            return Boolean.class;
610                    } else if (c == Character.class) {
611                            return char.class;
612                    } else if (c == char.class) {
613                            return Character.class;
614                    } else if (c == Byte.class) {
615                            return byte.class;
616                    } else if (c == byte.class) {
617                            return Byte.class;
618                    } else if (c == Short.class) {
619                            return short.class;
620                    } else if (c == short.class) {
621                            return Short.class;
622                    } else if (c == Long.class) {
623                            return long.class;
624                    } else if (c == long.class) {
625                            return Long.class;
626                    } else if (c == Float.class) {
627                            return float.class;
628                    } else if (c == float.class) {
629                            return Float.class;
630                    } else if (c == Double.class) {
631                            return double.class;
632                    } else if (c == double.class) {
633                            return Double.class;
634                    } else {
635                            return null;
636                    }
637            }

639            /**
640             * Returns a field from the given object that goes by the name of <code>fieldName</code>. If <code>o</code> is a Class object, a static
641             * field will be returned.
642             * 
643             * @param o The reference to the object to fetch the property value from.
644             * @param fieldName The identifier or name of the member field/property.
645             * @return The value of the <code>Field</code> or in case of a Dynamic the property value.
646             */
647            public static Field solveField(final Object o, final String fieldName) {
648                    try {
649                            if (o.getClass().equals(Class.class)) {
650                                    // Java static field
651                                    return ((Class<?>) o).getField(fieldName);
652                            } else {
653                                    // Java instance field
654                                    return o.getClass().getField(fieldName);
655                            }
656                    } catch (NoSuchFieldException e) {
657                            return null;
658                    }
659            }

661            /**
662             * Retrieves a {@link Method} from a cache.
663             * 
664             * @param datatype The owning {@link Class} of the <code>Method</code> being searched for.
665             * @param method The name of the method that is being searched for.
666             * @param signature The parameter list of the method we need to match if a method was found by name.
667             * @return The <code>Method</code> found on the specified owner with matching name and signature.
668             * @see JReflect#methodCache
669             * @see JReflect#addMethodToCache(Class, String, AccessibleObject, Class...)
670             */
671            private final static <T> AccessibleObject getMethodFromCache(final Class<T> datatype, final String method, final Class<?>... signature) {
672                    final Map<String, Map<AccessibleObject, Class<?>[]>> owner = methodCache.get(datatype);
673                    // we know only methods with parameter list are stored in the cache
674                    if (signature.length > 0) {
675                            // get owner, its methods matching specified name and match their signatures
676                            if (owner != null) {
677                                    final Map<AccessibleObject, Class<?>[]> signatures = owner.get(method);
678                                    if (signatures != null) {
679                                            for (final Map.Entry<AccessibleObject, Class<?>[]> entry : signatures.entrySet()) {
680                                                    if (Arrays.equals(entry.getValue(), signature)) {
681                                                            return entry.getKey();
682                                                    }
683                                            }
684                                    }
685                            }
686                    }
687                    // method not found or known not to be stored due to absent parameter list
688                    return null;
689            }

691            /**
692             * Adds a specific <code>Method</code> to the cache.
693             * 
694             * @param datatype The <code>Class</code> that owns the <code>Method</code>.
695             * @param method The <code>Method</code>'s name by which methods can be found on the specified owner.
696             * @param methodRef The <code>Method</code> reference that's actually being stored in the cache.
697             * @param signature The parameter list of the <code>Method</code> being stored.
698             * @see JReflect#methodCache
699             * @see JReflect#getMethodFromCache(Class, String, Class...)
700             */
701            private final static void addMethodToCache(final Class<?> datatype, final String method, final AccessibleObject methodRef,
702                            final Class<?>... signature) {
703                    // only store methods with a parameter list
704                    if (signature.length > 0) {
705                            // get or create owner entry
706                            Map<String, Map<AccessibleObject, Class<?>[]>> owner = methodCache.get(datatype);
707                            owner = owner != null ? owner : new LinkedHashMap<String, Map<AccessibleObject, Class<?>[]>>();
708                            // get or create list of methods with specified method name
709                            Map<AccessibleObject, Class<?>[]> methods = owner.get(method);
710                            methods = methods != null ? methods : new LinkedHashMap<AccessibleObject, Class<?>[]>();
711                            // add or overwrite method entry
712                            methods.put(methodRef, signature);
713                            // finally shelve all the stuff back
714                            methods.put(methodRef, signature);
715                            owner.put(method, methods);
716                            methodCache.put(datatype, owner);
717                    }
718            }

720            /**
721             * Assigns a value to a field <code>id</code> on the given object <code>o</code>. If a simple assignment fails, a common conversion will
722             * be attempted.
723             * 
724             * @param o The object to find the field on.
725             * @param property The name of the field we're assigning the value to.
726             * @param value The value to assign to the field, may be converted to the field's type through common conversion.
727             * @return The actual value that was assigned (the original or the converted value).
728             * @throws IllegalAccessException Thrown by {@link Field#set(Object, Object)}
729             * @throws NoSuchFieldException
730             * @see ValueConverter#convert(Object, Class)
731             */
732            public static Object assignToField(final Object o, final String property, final Object value)
733                            throws IllegalAccessException, NoSuchFieldException {
734                    Object assignedValue = value;
735                    final Field field = solveField(o, property);
736                    if (field != null) {
737                            try {
738                                    field.set(o, value);
739                            } catch (final IllegalArgumentException ie) {
740                                    assignedValue = ValueConverter.convert(value, field.getType());
741                                    field.set(o, assignedValue);
742                            }
743                            return assignedValue;
744                    } else {
745                            throw new NoSuchFieldException();
746                    }
747            }

760             * Returns the smallest class that can hold all of the specified numbers.
761             * 
762             * @param numbers The list with numbers that all should fit in the <code>Number</code> container.
763             * @return The <code>Number</code> container that is just large enough for all specified numbers.
764             */
765            public static Class<?> widestNumberClass(final Number... numbers) {
766                    // find widest number
767                    Integer widest = 0;
768                    Class<?> widestNumberType = Byte.class;
769                    for (final Number number : numbers) {
770                            final Integer size = numSizes.get(number.getClass());
771                            if (size > widest) {

774                            }
775                    }
776                    return widestNumberType;

781             * 
782             * @param subject The <code>Object</code> who's properties/fields need to be reflected.
783             * @return A list of names that represent the fields on the given <code>Object</code>.
784             */
785            public static Collection<String> collectProperties(final Object subject) {
786                    final Collection<String> properties = new LinkedHashSet<String>();
787                    // collect properties/fields
788                    final Field[] fields = subject.getClass().getFields();
789                    for (final Field f : fields) {

798             * 
799             * @param subject The <code>Object</code> or <code>Dynamic</code> who's methods need to be reflected.
800             * @return Returns a list with methods, either {@link Method}s, or <code>Dynamic</code> methods.

803                    final Collection<String> methods = new LinkedHashSet<String>();
804                    // collect methods
805                    final Method[] methodlist = subject.getClass().getMethods();
806                    for (final Method m : methodlist) {
807                            methods.add(m.getName());

809                    return methods;
